# ブロック崩しゲーム

HTML5 Canvas で作成したブロック崩しゲームです。

## ゲーム概要

- **ジャンル**: アクション/パズル
- **プレイ人数**: 1人
- **対応デバイス**: PC、スマートフォン、タブレット

## 遊び方

### 目的
すべてのブロックを壊してクリアを目指そう！

### 操作方法

#### PC
- **← → キー**: パドルを左右に移動
- **マウス**: パドルを移動（カーソル追従）
- **スペースキー / クリック**: ボール発射（ゲーム開始時）

#### スマートフォン/タブレット
- **タッチ**: パドルを移動（指の位置に追従）
- **タップ**: ボール発射

### ルール

1. パドルでボールを跳ね返してブロックを壊す
2. ボールを落とすと残機が1つ減る
3. 残機が0になるとゲームオーバー
4. すべてのブロックを壊すとゲームクリア

### スコア

- ブロック1個破壊: **10点**

## カスタマイズ方法

### 難易度調整

`game.js` の以下の値を変更：

```javascript
// ボール速度を上げる
const ball = {
    speed: 4,  // この値を大きくする（例: 6）
    dx: 4,     // 横方向の速度
    dy: -4     // 縦方向の速度
};

// パドル幅を変更
const paddle = {
    width: 100, // この値を変更（小さくすると難しい）
    height: 15,
    speed: 8    // 移動速度
};

// 残機数を変更
let lives = 3; // 初期残機数
```

### ブロック配置変更

```javascript
const brickInfo = {
    rows: 5,    // 行数を変更
    cols: 8,    // 列数を変更
    width: 90,  // ブロック幅
    height: 25, // ブロック高さ
    padding: 10 // ブロック間隔
};
```

### 色変更

// ブロックの色
